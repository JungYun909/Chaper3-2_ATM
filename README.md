## 📌주요 기능
- 팝업 형식으로 하나의 씬에서 진행
- 입출금 금액이 잔액보다 높을 때, 잔액 부족 팝업창
- 버튼을 통해 입출금액을 정하거나, 직접 입력하여 진행
